# File Organizer

Organizes files in a directory by type, moving them into categorized subfolders.

## Categories

| Folder     | Extensions                        |
|------------|-----------------------------------|
| Images     | .jpg .jpeg .png .gif .webp        |
| Documents  | .pdf .docx .txt .xlsx             |
| Videos     | .mp4 .mkv .avi                    |
| Audio      | .mp3 .wav .flac                   |
| Zip        | .zip .rar .7z .iso                |
| Others     | .exe .java .py .ipynb             |

## Install

```bash
curl -sSL https://github.com/Willian-Fagundes/file-organizer/blob/main/organizer.py| bash
```

## Usage

Navigate to the folder you want to organize and run:

```bash
cd ~/Downloads
organizar
```

## Requirements

- Python 3.8+
- Git or curl
